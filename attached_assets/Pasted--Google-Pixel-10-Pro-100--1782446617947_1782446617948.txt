# جعل محاكاة Google Pixel 10 Pro واقعية 100%

المشكلة الحالية: المحاكاة تستخدم بيانات قديمة/غير دقيقة (Chrome 124, Build ID خاطئ, أبعاد شاشة خاطئة, لا يوجد User-Agent Client Hints, لا يوجد حماية من الكشف). هذا يجعل Google تكتشف أن الجهاز ليس حقيقياً.

## التغييرات المطلوبة

### 1. تحديث المواصفات في `config.py`

| القيمة | الحالية (خاطئة) | الجديدة (حقيقية) |
|---|---|---|
| `BUILD_ID` | `AP4A.250405.002` | `CP1A.260405.005` |
| `CHROME_VERSION` | `124.0.6367.82` | `149.0.7827.200` |
| `CHROME_MAJOR_VERSION` | `124` | `149` |
| Screen size | `390x844` | `412x915` (CSS viewport) |
| Pixel ratio | `3.0` | `3.5` (~495 PPI) |
| User-Agent format | يحتوي على model name | الشكل الحديث: `Android 10; K` (Chrome UA reduction) |

> [!IMPORTANT]
> Chrome منذ الإصدار 110+ يستخدم **User-Agent Reduction** — أي أن الـ User-Agent الحقيقي لا يحتوي على اسم الجهاز. الشكل الصحيح هو:
> ```
> Mozilla/5.0 (Linux; Android 10; K) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/149.0.7827.200 Mobile Safari/537.36
> ```
> Google تعتمد على **Client Hints** (`Sec-CH-UA-Model`, `Sec-CH-UA-Platform`) لتحديد الجهاز.

### 3. تحسين `google_automation.py` — `_build_driver()`

التغييرات الرئيسية:

#### أ. أبعاد الشاشة الحقيقية
```python
# Pixel 10 Pro: 1280x2856 native, CSS viewport ~412x915 @3.5x
"deviceMetrics": {"width": 412, "height": 915, "pixelRatio": 3.5}
```

#### ب. إضافة CDP commands بعد إنشاء الـ driver:

1. **`Emulation.setUserAgentOverride`** — مع `userAgentMetadata` الكامل
2. **`Page.addScriptToEvaluateOnNewDocument`** — لإخفاء `navigator.webdriver`
3. **إخفاء بصمة الأتمتة** — عبر JavaScript injections شاملة:
   - إخفاء `navigator.webdriver`
   - تزييف `navigator.platform` → `"Linux armv8l"`
   - تزييف `navigator.hardwareConcurrency` → `8` (Tensor G5)
   - تزييف `navigator.deviceMemory` → `16` (16GB RAM)
   - تزييف `navigator.maxTouchPoints` → `5`
   - إضافة Touch support flags
   - تزييف WebGL renderer → `"PowerVR DXT-48-1536"`

#### ج. إضافة Chrome flags إضافية للواقعية:
```python
options.add_argument("--enable-features=NetworkService,NetworkServiceInProcess")
options.add_argument("--lang=en-US")
```
